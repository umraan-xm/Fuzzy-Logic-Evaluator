# Fuzzy-Logic-Evaluator
A desktop application that evaluates students using Fuzzy Logic techniques. This application is an implementation of the following research paper:
https://www.researchgate.net/publication/261397611_Application_of_Fuzzy_Logic_for_Performance_Evaluation_of_Academic_Students
